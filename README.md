# Conqueror Routes (Multi-challenge PWA)

## What it does
- Track **multiple active missions** at the same time.
- Each mission has its own **timeframe** (start date + days).
- You can log posts to **any** active mission.

## Deploy on GitHub Pages
1. Create a GitHub repo (e.g., `conqueror-routes`).
2. Upload the contents of this zip to the repo root:
   - `index.html`
   - `manifest.json`
   - `sw.js`
   - `icons/`

3. GitHub → **Settings → Pages**
   - Source: **Deploy from a branch**
   - Branch: `main`, folder: `/root`

## Install on Android
Open the GitHub Pages URL in Chrome → **⋮** → **Install app**
